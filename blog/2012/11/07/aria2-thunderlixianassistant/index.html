下载利器 -- Aria2 + 迅雷离线助手

运行环境: Ubuntu Desktop 12.10 32bit

安装aria2
$ sudo apt-get install aria2

设置Ubuntu开机自动运行Aria2 $ sudo vi /etc/init.d/Aria2.sh

$ sudo chmod +x /etc/init.d/Aria2.sh
$ sudo update-rc.d /etc/init.d/Aria2.sh defaults
